# MPU (SAM / general) — Click board bundles

This directory is reserved for **MikroE Click board** integration bundles targeting **32- and 64-bit MPU** platforms built with Microchip **`meta-mchp-mpu`** (see [linux4microchip/meta-mchp](https://github.com/linux4microchip/meta-mchp)).

Add new bundles here as subdirectories (each with its own `README.md`, overlays, patches, and scripts as needed). Paths and recipes will differ from PolarFire SoC bundles—copy patterns from existing bundles rather than reusing their patches verbatim.

## Parent repository

See the [repository root `README.md`](../README.md) for overall layout and how bundles are organized by platform.
