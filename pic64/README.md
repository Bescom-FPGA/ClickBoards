# PIC64 MPU — Click board bundles

This directory is reserved for **MikroE Click board** integration bundles targeting **PIC64** evaluation platforms built with Microchip **`meta-mchp-pic64`** (see [linux4microchip/meta-mchp](https://github.com/linux4microchip/meta-mchp)).

Add new bundles here as subdirectories (each with its own `README.md`, overlays, patches, and scripts as needed).

## Parent repository

See the [repository root `README.md`](../README.md) for overall layout and how bundles are organized by platform.
